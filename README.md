# MSP430G2-Geiger-Counter

This device is Geiger-Muller Counter ,and Random Number Generator(RNG) uses radioactive isotopes.

## Hardware

### Key Component List
- Geiger-Muller Tube
- MSP430G2433 or MSP430G2553
- TLC555
